1.
En primer lugar hay que situarse en la carpeta que queremos que se añada el archivo Angular, posteriormente utilizar el comando "ng new".
Este comando es para crear un proyecto, al pulsar enter nos preguntará por el nombre del proyecto, también nos pregunta si queremos añadir 
las rutas de angular en mi caso no, y por el tipo de estilo en mi caso css.

2.
En segundo lugar hay que crear los componentes para ello se requiere el siguiente comando: "ng g component components/mostrar_plantas". 
La "g" es una abreviatura de generate también se podría escribir así "ng generate component components/mostrar_plantas".

3.
Para finalizar usaremos este comando: “ng serve”, este comando se utiliza para arrancar el servicio de Angular.
